# 🐍 Snake Verse

Snake Verse is a Flask-based web application that identifies snake species from an uploaded image using a trained deep learning model. The application provides a simple and user-friendly interface for image classification.

## 🚀 Features

* Upload a snake image
* AI-powered snake species prediction
* Fast and easy-to-use web interface
* Built with Flask and TensorFlow/Keras

## 🛠️ Tech Stack

* Python
* Flask
* TensorFlow / Keras
* HTML, CSS, JavaScript

## 📂 Installation

```bash
git clone https://github.com/prashant-singh-78/snake-verse.git
cd snake-verse
pip install -r requirements.txt
python app.py
```
